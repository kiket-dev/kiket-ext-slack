# Changelog

All notable changes to the Slack Notifications extension.

## [1.0.0] - 2025-01-01

### Added
- Initial release
- Send messages to channels and direct messages
- Rich formatting with mrkdwn, markdown, and HTML support
- Threaded replies
- Message attachments
- Interactive approval requests with buttons
- SLA breach notifications
- Customizable Liquid message templates
- Channel and user validation endpoints
- Automatic rate limit handling with retry-after
